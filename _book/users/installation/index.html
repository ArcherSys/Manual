How to install ArcherSys OS

==========

  1. Download The Tools
    1. XAMPP
    2. Git
    3. Chrome or Firefox
  2. Install XAMPP and Git
  3. Install RailsInstaller and Python

  4. Install nodejs-portable if not already.

    You will need this for Gitbooks.io to be an author of this book

  6. Type cd <PATH>/htdocs
  7. Type git init
  8. Type

!FILENAME //github.com/ArcherSys/ArcherSys git remote add origin https

  • Type git pull origin master
  • Register for ArcherSys OS
  • Download any Applications at the ArcherSys OS Apporium
